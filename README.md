# bible-vue

A simple front-end Vue project for consuming a Holy Bible API.

## Setup

```
npm  install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
